Chicago Data Analysis with SQL & Python
This project analyzes Chicago’s public datasets to uncover insights on socioeconomic indicators, crime patterns, and school safety. Using Python, pandas, and SQLite, the datasets are loaded into a database and explored through SQL queries in a Jupyter Notebook.

Datasets Used
Socioeconomic Indicators in Chicago – Community area-level data on poverty, hardship index, and other indicators.

Chicago Public Schools – School performance and safety scores.

Chicago Crime Data – Reported crimes in Chicago from 2001 to present.

Key Analysis & Queries
Identify community areas with the highest poverty rates.

Find the most crime-prone areas and crime types in schools.

Determine community areas with the highest hardship index.

Calculate average safety scores by school type.

Skills Demonstrated
Python & pandas for data cleaning and loading

SQLite & SQL queries for data analysis

Jupyter Notebook for interactive exploration and reporting
